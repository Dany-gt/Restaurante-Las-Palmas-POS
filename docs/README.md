# Documentación del Proyecto: Restaurante Las Palmas POS

Este directorio contiene la documentación detallada de los módulos del sistema POS, incluyendo su funcionamiento, categorías y esquemas de base de datos SQL.

## Estructura de Módulos

- [📦 Inventarios](modules/inventarios.md): Gestión de insumos, stock, compras y kardex.
- [🍽️ Menú de Platillos](modules/menu_platillos.md): Administración de platillos, recetas, opciones y modificadores.
- [📊 Contabilidad / SAT](modules/contabilidad.md): Sincronización con SAT, auditoría de facturas, libros contables e impuestos.
- [💰 Ventas / POS](modules/ventas_pos.md): Terminal de punto de venta, gestión de órdenes, cajas y turnos.
- [📄 Cuentas por Cobrar](modules/cuentas_por_cobrar.md): Gestión de créditos y estados de cuenta de clientes.
- [⚙️ Configuración del Sistema](modules/configuracion.md): Ajustes globales, impresoras, roles y usuarios.
