# Project Brief: Restaurante Las Palmas POS

Sistema de Punto de Venta (POS) integral diseñado específicamente para el Restaurante Las Palmas. El sistema abarca desde la toma de pedidos hasta la gestión de producción en cocina e inventarios.

## Goals
- Optimizar la velocidad de servicio en el restaurante.
- Proporcionar una interfaz técnica de alta densidad para la cocina (Módulo de Producción).
- Asegurar la integridad de datos mediante sincronización con Supabase.
- Mantener una estética premium y funcional bajo el estándar "Antigravity OS".

## Scope
- Mantenimiento de Platillos e Insumos.
- Gestión de Inventarios y Kardex.
- Monitoreo de Producción en tiempo real con cronómetros y fichas técnicas.
- Sincronización de facturación con el SAT.
